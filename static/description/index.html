HR Meal Management

This module provides a complete meal management system integrated with Human Resources. It allows organizations to configure meal packages, assign meals to employees, calculate daily and periodic meal consumption, track expenses.

Key Features

  • Define multiple meal packages (e.g., staff, officers, management)
  • Configure meal types such as breakfast, lunch, dinner
  • Group employees by meal schedules using meal groups
  • Assign meals and meal groups to employees
  • Calculate meals for single dates or date ranges
  • Daily and periodic meal reporting
  • Allocate meal expenses and calculate employee meal cost

Configuration

1. Meal Packages

Meal packages represent different categories of meals. For example, officers and staff may have different meal compositions or costs.

2. Meal Types

Meal types define individual meals such as breakfast, lunch, or dinner. Each meal type can be linked to a meal package and configured with weightage and subsidy rules.

3. Meal Groups

Meal groups are used to categorize employees based on their meal schedules. For example:

  • Daily meals
  • Weekdays only
  • Excluding weekends and holidays

4. Meal Assignment

Assign meal types and meal groups to employees. Each employee can be assigned once to avoid duplication.

Meal Calculation

Meals can be calculated for:

  • A single day
  • A specific date range
  • Selected employees or all employees
  • A selected meal group or all groups
Note: Default meals are generated automatically based on configuration. Any skipped or additional meals can be manually adjusted after calculation. Bulk updates are supported for leave days or schedule changes.

Daily Meals

The Daily Meals menu provides a detailed view of meal records. Users can review, edit, and correct meal entries for any date or date range.

Meal Expense

Meal-related expenses can be recorded for a specific date range. Expenses can be:

  • Linked to a specific meal package
  • Or distributed evenly across all meals if no package is selected

Employee Meal Cost

The system calculates:

  • Total number of meals
  • Meal rate
  • Total meal expense per employee

Target Users

  • HR Officers
  • HR Managers
  • Accounts & Payroll Teams
